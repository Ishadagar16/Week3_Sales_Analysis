# Week 3 – Sales Data Analysis Project

## Project Overview

This project is part of Week 3: Introduction to Data Analysis.  
The objective of this project is to analyze a real-world sales dataset using Python and the Pandas library.

The dataset contains 100 sales transactions from January to April 2024.  
The project performs data exploration, data cleaning, and business analysis to generate useful insights.

## Project Objectives

- Load CSV data using pandas
- Explore dataset structure
- Clean missing or duplicate values
- Calculate important business metrics
- Generate a formatted sales report
- Organize project files properly


## Technologies Used

- Python 3
- Pandas Library


## Setup Instructions

### 1. Install Python

Download and install Python from:  
https://python.org  

Make sure to check "Add Python to PATH" during installation.

---

### 2. Install Required Library

Open Command Prompt and run:

pip install pandas

### 3. Run the Project

Navigate to the project folder and run:

```
python sales_analysis.py
```

The program will display:
- Dataset details
- Data cleaning information
- Final sales analysis report

## Dataset Information

The dataset contains:

- 100 rows
- 7 columns:
  - Date
  - Product
  - Quantity
  - Price
  - Customer_ID
  - Region
  - Total_Sales

Products included:
- Phone
- Laptop
- Tablet
- Monitor
- Headphones

Regions included:
- East
- West
- North
- South

## Data Cleaning Performed

- Checked for missing values
- Removed duplicate rows
- Converted Date column to datetime format

No missing values were found in the dataset.

## Analysis Performed

The following metrics were calculated:

- Total Revenue
- Average Sale Value
- Highest Single Sale
- Lowest Single Sale
- Best-Selling Product (based on quantity)
- Region-wise Sales
- Monthly Sales

## Key Findings

- The dataset generated high total revenue.
- Laptop was identified as the best-selling product.
- South region generated the highest revenue.
- January recorded the highest monthly sales.
